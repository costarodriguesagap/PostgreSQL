\l 			- lista as base de dados existentes
\c <database_name> 	- conectar a uma determinada base de dados
\d 			- mostra o detalhe da base de dados
\d <table_name> 	- mostra a definição de uma determinada tabela
\i <path_file>          - executa comandos SQL contidos no ficheiro localizado no <path_file> (usar o site https://www.mockaroo.com/ para gerar dados para inserir numa tabela)
